详情介绍

  • 掌柜钱包是一款怎样的理财产品

    掌柜钱包是由兴业银行股份有限公司与兴业全球基金管理有限公司在互联网金融平台“钱大掌柜”上联手推出的新型现金管理工具，对接兴全添利宝货币市场基金。

    1. 高收益
    -余额理财收益稳居第一

    2. 低门槛
    -最低1分钱起购，转入金额无上限
    -拥有业内最高赎回上限3000万
    -打破时间限制，随存随取，快速取现

    3. 无风险
    -资金只在同张银行卡内购买和赎回
    -整个系统的安全性由兴业银行保障
    -多种安全措施，短信口令、密码加密、操作超时保护等


    4. 超便捷
    -支持近百家银行卡、折购买

  • 掌柜钱包能不能保本？

    掌柜钱包是钱大掌柜推出的一项余额增值服务，用户把资金转入掌柜钱包，实际是购买了一款由兴业全球基金提供的名为添利宝的货币基金。主要投资于短期货币工具如国债、中央银行票据等。具有风险低、收益稳定特点，素有“活期存款最佳替代品”之称，但理论上有损失的风险存在。

  • 掌柜钱包的收益比理财产品高那么多，谁还买理财？

    您好，用户转入掌柜钱包的钱实际是购买了一款货币基金，货币基金是投资于长期存款、债券等相关金融工具，金融工具的价格会有所波动，因此用户每天的收益也会有变化。您可以根据您对流动性、收益性、期限等的偏好选购不同的理财产品。

  • 购买掌柜钱包有手续费吗？

    购买不收任何手续费.

  • 可以用别人的银行卡购买吗？

    不可以。购买掌柜钱包的银行卡必须是本人的，不支持用他人银行卡购买。

  • 掌柜钱包这个可以在柜面上办吗？

    掌柜钱包是兴业银行与兴业全球基金联手推出的新一代余额理财工具，目前用户可以通过钱大掌柜网页、手机端进行购买。钱大掌柜是掌柜钱包唯一销售渠道。

收益

  • 什么时候能看到我赚的钱啊？

    每天15点前转入掌柜钱包的资金，第二个工作日开始计算收益，遇到节假日顺延，计算收益后的第二天开始发放收益。转出资金不计算当日收益，优先转出已确认的资金。
    转入时间 首次计收益时间 首次看到收益时间
    周一15:00～周二15:00 周三 周四
    周二15:00～周三15:00 周四 周五
    周三15:00～周四15:00 周五 周六
    周四15:00～周五15:00 下周一 下周二
    周五15:00～下周一15:00 下周二 下周三
    双休日及国家法定假期，基金公司不进行份额确认，需顺延到下一个工作日确认。

  • 钱包中显示的收益是当天的吗？每天结算吗？

    不是当天的，收益上方会显示收益获得的具体时间。收益是每天结算的，更新时间一般在早上4点左右.

  • 各个收益的计算是怎么得出来的？

    万分收益=（基金当天总收益—基金费用）*10000/基金总份额
    昨日收益=总资产（已确认份额）*万分收益/10000
    累计收益=从开通掌柜钱包起所有收益的总和

  • 购买金额太少就没有收益吗？

    当投资人投资金额按万份收益计算后，收益不足1份，无法获得收益分配。从投资的角度，建议购买100元以上。

  • 掌柜钱包是复利吗？（请问掌柜钱包产生的收益是和本金在一起利滚利，还是收益和本金分开存放？）

    是的。掌柜钱包是复利计息。

转出

  • 我急需用钱，钱包转出有金额限制吗？

    土豪，为了凸显您的尊亏，您的单日单笔和累计赎回金额有3000 万！单日最多转出3 次哦。

  • 我按【转出】了，什么时候钱能到账？

    如果您绑定的是兴业或合作银行卡，钱包转出资金“咻”的一下马上到银行的活期账户上了；如果您绑定的是银联卡， 钱包转出资金是“咻”的一下马上到理财账户上，注意是理财账户哦，您可以将理财账户资金转至相应银行卡内。注意每日转出限额哦~

  • 钱包转出当天有收益吗？

    没有，转出当天是没有收益的。
